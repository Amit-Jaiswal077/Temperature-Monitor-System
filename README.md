# Temperature-Monitor-System
“An Arduino‑based Temperature Monitor System that measures ambient temperature using sensors and displays real‑time readings. Designed for accuracy and simplicity, it helps track environmental changes, trigger alerts, and can be extended for IoT or smart home applications.”
# Component Used:
Arduino Uno R3
Temperature Sensor [TMP36]
LCD 16 x 2
250 kΩ Potentiometer
